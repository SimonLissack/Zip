Zips sub-folders of the selected folder in bulk.

Admin rights may be needed due to registry access (To retrieve file extensions)

Requires Visual Studio 2010

To zip folders:
	1. Press "Browse" and navigate to the folder containing the folders to be zipped
	2. Select folders from the list (multiple folders can be selected using ctrl + left mouse button)
	3. (optional) To only zip certain file types, press "Filter" and select file types from the list
	4. Press "Zip Selected", when the progress bar is full, a sound will play, indicating that the zipping has finished
